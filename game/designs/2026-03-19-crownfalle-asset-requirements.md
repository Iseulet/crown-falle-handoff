# CrownFalle Proto — 에셋 & 프레임워크 요구사항

> 날짜: 2026-03-19
> 상태: 1차 정리 (TBSF 데모 체험 후 갱신 예정)

## 1. 개요

Unity 6.3 LTS + TBSF 기반 2D 턴제 전술 RPG 개발에 필요한 에셋, 프레임워크, 도구를 정리한다.
4-Layer 게임 루프(Town/Camp → WorldMap → Battle → Roster) 기준으로 분류.

### 태그 범례

| 태그 | 의미 |
|------|------|
| 🟢 보유 | 이미 구매/설치 완료 |
| 🔴 필수 | 없으면 해당 레이어 구현 불가 |
| 🟡 권장 | 개발 효율 또는 퀄리티 유의미하게 향상 |
| 🔵 선택 | 프로토타입 이후 추가해도 무방 |

## 3. Layer 1: Town / Camp (거점)

### 3-1. 대화 / 내러티브 시스템

TBSF 내장 대화 시스템 확인 후 최종 결정. 아래는 외부 시스템 후보 3종.

| 태그 | 항목 | 특징 | 가격 | 적합도 |
|------|------|------|------|--------|
| 🔴 후보① | Pixel Crushers Dialogue System | 업계 표준. 노드 에디터, 퀘스트, 세이브/로드, 컷씬 통합. Disco Elysium/Last Epoch에서 사용. 100+ 에셋 연동 | ~$65 | 높음 — 올인원. 학습 곡선 있음 |
| 🔴 후보② | Yarn Spinner | 오픈소스(MIT). 텍스트 기반 작성, Git diff 친화적, Speech Bubble/Dialogue Wheel 애드온 | 무료 | 높음 — 가볍고 솔로 개발 친화적 |
| 🔴 후보③ | Ink (by Inkle) | JSON 컴파일, 작가 친화적 문법, Pixel Crushers와 하이브리드 가능 | 무료 | 중간 — UI는 직접 구현 필요 |

**결정 기준**:
- TBSF 내장 대화가 분기/조건부 선택지를 지원하면 → 내장 사용 (추가 비용 없음)
- 내장이 단순 텍스트 표시 수준이면 → Yarn Spinner(무료) 우선 검토
- 퀘스트/세이브 통합까지 필요하면 → Pixel Crushers Dialogue System

### 3-2. UI

| 태그 | 항목 | 용도 | 가격 | 비고 |
|------|------|------|------|------|
| 🟢 | uGUI + TextMeshPro | 기본 UI 프레임워크 | 무료 | Unity 내장 |
| 🟡 | Dark Fantasy GUI / UI Kit | 다크 중세 판타지 UI 스킨 (버튼, 패널, 프레임, 슬롯) | ~$15~25 | 프로토타입 이후 교체 가능 |
| 🔵 | Modern UI Pack | 범용 UI 컴포넌트 (드롭다운, 슬라이더, 모달) | ~$30 | uGUI 보충용, 필수는 아님 |

### 3-3. 인벤토리 / 장비

| 태그 | 항목 | 용도 | 가격 | 비고 |
|------|------|------|------|------|
| 🔵 | Item & Inventory System (Devion Games) | 드래그&드롭 인벤토리, 장비 슬롯, 크래프팅 | ~$40 | 직접 구현 대비 시간 절약. 무기 기반 정체성과 맞는지 확인 필요 |
| 🔵 | 직접 구현 (ScriptableObject 기반) | 커스텀 인벤토리 | 무료 | 기존 JSON 데이터 구조와 완전 호환 보장 |

**결정 기준**: 장비 슬롯이 단순(머리/갑옷/장갑/신발/악세/양손무기)하면 직접 구현이 나을 수 있음. 복잡한 드래그&드롭/스택/필터가 필요하면 에셋 활용.

## 4. Layer 2: World Map (여행/탐험)

| 태그 | 항목 | 용도 | 가격 | 비고 |
|------|------|------|------|------|
| 🔴 | 월드맵 일러스트 | 포인트 클릭 맵 배경 이미지 | 미정 | 커미션 / AI 생성 / 직접 제작. 게임의 첫인상을 결정하는 핵심 아트 |
| 🔴 | 이벤트 시스템 | 랜덤 이벤트 + 텍스트 선택지 처리 | — | 대화 시스템과 통합 구현 (별도 에셋 불필요) |
| 🟡 | 맵 아이콘/마커 팩 | 마을/던전/이벤트/상인 위치 표시 | ~$5~15 | 2D 아이콘 세트. itch.io에 무료/저가 다수 |
| 🔵 | 미니맵 UI | 현재 위치/경로 표시 | — | 포인트 클릭이라 미니맵 불필요할 수도 있음 |

**참고**: 포인트 클릭 월드맵은 구현 복잡도가 가장 낮은 레이어. 핵심은 일러스트 퀄리티와 이벤트 콘텐츠.

## 9. 우선순위 구매/준비 순서

### Phase 0 (TBSF 체험 중) — 비용 $0

1. ✅ Newtonsoft Json.NET 설치
2. ✅ Unity MCP 설치 + Claude Code 연동 확인
3. ✅ DOTween 무료 코어 설치
4. ✅ TBSF 데모 체험 → 내장 대화/이벤트 시스템 확인
5. ✅ 플레이스홀더 스프라이트 수집 (Kenney, itch.io CC0)

### Phase 1 (데이터 이식) — 비용 $0

- 추가 에셋 불필요. Newtonsoft + TBSF만으로 진행.

### Phase 2~3 (전투 포팅) — 비용 ~$55 (선택)

- Odin Inspector 구매 고려 (ScriptableObject 작업 효율)
- 전투 HUD는 uGUI + TMP로 직접 구현

### Phase 4 (비주얼 + UI) — 비용 ~$50~100

- 대화 시스템 최종 결정 + 구매
- Dark Fantasy GUI Kit 구매
- 초상화/아이콘 팩 구매

### Phase 5 (검증 후) — 필요에 따라

- 2D 스프라이트 통합 팩 (최종 아트 방향 확정 후)
- BGM/SFX (분위기 확정 후)
- 전투 이펙트/배경 (재미 검증 후)
