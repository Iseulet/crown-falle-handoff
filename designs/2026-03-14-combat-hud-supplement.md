## 갭 1 (높음): 교전 이탈 모달 — 4종 미반영

### 문제

이탈 4종(강제/VP/스킬/아군지원) 중 모달에 2종(VP이탈/강제이탈)만 있음.
스킬 이탈(연막)과 아군 지원 이탈(도발)은 별도 경로.

### 보강

DisengageModal을 **교전 중 이동 시도 시**만 표시하고,
스킬/아군지원 이탈은 스킬 바에서 직접 처리.

```
교전 중 이동 시도:
  → DisengageModal 표시:
    [VP 이탈]     VP 1 소모, 기회 공격 없음   (VP≥1 시 활성)
    [강제 이탈]   기회 공격 받음
    [취소]

교전 중 스킬 사용:
  연막(슬롯 3) 클릭 → 스킬 실행 → 교전 자동 해제 (모달 없음)
  → 스킬 바에 "[교전 해제]" 뱃지가 연막 슬롯에 표시

아군 지원 이탈:
  다른 아군 선택 → 도발 스킬 사용 → 교전 대상 전환 (모달 없음)
  → 타겟팅 시 교전 중인 적을 선택하면 "교전 전환" 예고 표시
```

### SkillSlot 교전 해제 뱃지

교전 해제 효과가 있는 스킬(연막, 그림자 일격 등)은 슬롯에 작은 뱃지:

```
┌──────────┐
│  3  🌫   │
│  연막    │
│    S4    │  ← 기존
│ [탈출!]  │  ← 교전 중일 때만 표시되는 뱃지
└──────────┘
```

## 갭 2 (높음): 교전 중 공격 대상 제한 표시

### 문제

교전 상태면 교전 상대만 공격 가능한데, 이 제한이 UI에 안 보임.

### 보강

교전 중일 때:
- 스킬 바 공격/스킬 슬롯은 활성 유지 (교전 상대에게 사용 가능)
- 타겟팅 시 **교전 상대만 빨간 하이라이트**, 다른 적은 하이라이트 없음
- 다른 적 호버 시 "⚠ 교전 상대만 공격 가능" 팝업
- 이동 슬롯은 비활성 (교전 이탈 필요)

```
교전 중 타겟팅:
  교전 상대 타일: 빨간 하이라이트 (공격 가능)
  다른 적 타일:   하이라이트 없음
  다른 적 호버:   "⚠ 교전 중 — 교전 상대만 공격 가능"
```

## 갭 3 (높음): 패시브 스킬 UI

### 문제

12종 패시브가 있지만 UI 어디에도 패시브 목록/발동 표시가 없음.

### 보강

유닛 상태 패널 하단에 패시브 영역 추가:

```
┌─ 선택 유닛 상태 패널 ──────────────────┐
│ ... (게이지, 상태이상 태그) ...          │
│                                        │
│ ── 패시브 ──                            │
│ ◆ 강인한 체력   ✓                      │  ← 발동 중 (밝은 색)
│ ◆ 방어 본능     ○ HP>50%              │  ← 대기 (어두운 색 + 조건)
│ ◆ 전열 유지     ✓                      │  ← 발동 중
└────────────────────────────────────────┘
```

### 표시 규칙

| 상태 | 표시 |
|------|------|
| 발동 중 (permanent/active) | ✓ + 밝은 텍스트 |
| 대기 (조건 미충족) | ○ + 어두운 텍스트 + 조건 요약 |
| 이번 턴 발동됨 | ✓ + 짧은 노랑 펄스 |

### 패시브 호버 툴팁

패시브 이름 호버 시 간략 툴팁:

```
┌─ 방어 본능 ──────────────────┐
│ on_hit, HP ≤ 50% 시 발동     │
│ 효과: ARM +10% (해당 턴)     │
│ 현재: 대기 (HP 78%)         │
└──────────────────────────────┘
```

### 공간 문제 해결

패시브 3개까지 직접 표시, 4개 이상이면 스크롤 또는 접기.
현재 직업당 최대 3개이므로 문제 없음. 향후 extra_passives 추가 시 접기 적용.

## 갭 4 (높음): 유닛 행동 완료 표시

### 문제

아군 페이즈에서 어떤 유닛이 이동/공격을 이미 했는지 구분 불가.

### 보강

#### 3D 전장 — 유닛 위 상태 표시

```
행동 가능:     파벌 서클 파란색 (기존)
이동 완료:     파벌 서클 파란색 → 절반 어두움 (반 행동 완료)
행동 완료:     파벌 서클 회색 (모든 행동 소진)
턴 대기 중:    파벌 서클 없음 (적 턴)
```

#### 유닛 상태 패널 — 행동 상태 뱃지

```
┌─ Aldric ──────────── [⚔] ─────────┐
│ Fighter Lv.3         [이동 ✓] [공격 ○] │
│                                        │
```

이동 ✓ = 이동 완료 (어두운 초록), 공격 ○ = 공격 가능 (밝은 빨강)

#### 스킬 바 연동

- `has_moved == true` → 이동 슬롯 비활성 (회색 오버레이)
- `has_attacked == true` → 공격/스킬 슬롯 전부 비활성

## 갭 5 (중간): 이동 STA 소모 예측

### 문제

이동 시 칸 수 × sta_cost_move 소모인데, 타겟 타일까지 몇 STA가 드는지 모름.

### 보강

이동 타겟팅 시 타일 호버:

```
타일 호버 → 경로 계산 → STA 소모 표시

3D 전장: 경로 위에 얇은 파란 선 + 도착 타일에 STA 소모량
         예: "3칸 → STA -3"

유닛 상태 패널 STA 바: 소모 예측을 빗금 영역으로 표시
         STA ████████████▒▒░░░░  17/28 (예측: -3)
                        ↑ 빗금 = 소모 예측
```

## 갭 6 (중간): 백어택 표시

### 문제

교전 방향 기준 후방 3칸에서 공격 시 백어택 보너스(CRIT +20%).
타겟팅 시 이 정보가 표시되지 않음.

### 보강

타겟팅 시 공격 가능 타일에 백어택 여부 표시:

```
일반 공격 가능 타일: 빨간 하이라이트
백어택 가능 타일:    빨간 + 노란 테두리 (또는 🗡 아이콘)

예상 결과 팝업에 백어택 보너스 표시:
  치명타  4.5% → 24.5% (+20% 백어택)
```

### 조건 (교전 시스템 기준)

```
교전 중인 적의 교전 방향(= 자신을 바라보는 방향) 기준
후방 3칸(hex) 위치에서 공격 시 백어택 성립

UI 표시:
  타겟 유닛 위에 "🗡 백어택" 플로팅 표시
  예상 결과 팝업: CRIT 보너스 +20% 명시
```

## 갭 7 (중간): 적 턴 행동 표시

### 문제

적 턴에 적이 뭘 하는지 플레이어가 인지할 방법 없음.

### 보강

적 턴 시:
- 행동 중인 적 유닛에 **빨간 파벌 서클** 표시 (기존 설계)
- 화면 상단에 행동 유닛 이름 + 행동 텍스트 표시:

```
┌──────────────────────────────────────┐
│ 🔴 용병단 낙오자 — 이동 → Aldric 공격│
└──────────────────────────────────────┘
```

- 적 행동 완료 후 0.5초 대기 → 다음 적 행동
- 카메라가 행동 중인 적을 자동 추적 (기존 soft_track 활용)

## 갭 8 (중간): 기회 공격 피드백

### 문제

강제 이탈 시 기회 공격이 발생하는데 시각 피드백 없음.

### 보강

```
강제 이탈 실행 시:
  1. 이동 유닛이 목적지로 이동 (이동 모션)
  2. 화면 중앙 상단: "⚔ 기회 공격!" 경고 배너 (1.5초)
  3. 교전 상대가 공격 모션 실행
  4. 데미지 숫자 플로팅 (빨간색, 큰 폰트)
  5. 카메라 shake_light
  6. 배너 사라짐 → 정상 흐름
```

### 경고 배너 스타일

```
┌─────────────────────────────────────┐
│  ⚔ 기회 공격!  용병단 낙오자 → Aldric │
│        데미지: 7  (ARM 흡수 5)       │
└─────────────────────────────────────┘
```

빨간 배경(30%) + 1.5초 페이드 아웃.

## 갭 9 (중간): STA 0 강제 대기 표시

### 문제

STA=0이면 이동/공격/스킬 전부 불가. 턴 종료만 가능. 명시적 경고 없음.

### 보강

STA 0 상태:
- 스킬 바 전체 비활성 (이동/공격/스킬 전부 회색)
- 턴 종료 버튼만 활성
- 유닛 상태 패널 STA 바 아래: "⚠ STA 소진 — 행동 불가" 경고
- 유닛 선택 시 3D 유닛 위: "💤" 아이콘

```
┌─ STA 소진 시 ──────────────────────┐
│ STA ░░░░░░░░░░░░░░░░░░  0/28      │
│ ⚠ STA 소진 — 행동 불가             │
└────────────────────────────────────┘
```

예외: Mage는 MP로 마법 스킬 사용 가능. MP 스킬 슬롯만 활성 유지.
